# JP Trust Learning — Public Research Library

Public-facing research library served at **https://public.jptrustlearning.com**

- `index.html` — single-file library landing (self-contained: brand fonts via Google Fonts, owl icon embedded as base64, no external assets)
- `robots.txt` / `sitemap.xml` — search/AI discoverability
- Deployed via **Cloudflare Pages** (auto-deploy on push to `main`)
- Separate from `jptrustlearning/pages` (member app) on purpose — this repo stays public.
